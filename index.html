Projects

Wireless Touchscreen Mouse

2014 - 2015

The main aim of the project is to design a Programmable Wireless Mouse which can be used for controlling the Mouse pointer in a Computer running windows 7/XP operating system. The project contains an Arduino which receives the input from a Resistive Touch screen and interacts with the computer using HC-05 bluetooth module. The commands received by the computer using a Bluetooth dongle are processed using a Python script which controls the computer mouse along with preprogrammed operations..

Arduino Smart Watch

2016 - 2017

Todayâ€™s cell phones have huge Touch screens which makes there usage uncomfortable when a person is Engaged in critical tasks like Driving. To bridge the gap and to ease the discomfort, a concept based Smart watch is designed using Arduino Pro Mini ,HC-05 bluetooth, OLED diplay and 3.7v 600mah Li-ion battery. A Smartphone App running on Android 4.0 connects to Arduino mini using bluetooth and provides the information to Arduino mini whenever a phone call/sms is received. The project can be extended to get GPS updates from Android Phone.

ADDING CPU

2017 - 2018

The main aim of the project is to design a simple Adding CPU using Von-Neumann Architecture in VHDL .In this project the CPU can perform instructions like Load, Store, Add and Jump.The data path and control unit are designed using VHDL and verified for operation using XILINX

Wireless PS/2 Keyboard

2017 - 2018

Due to the Outdated technology, nowadays PS/2 keyboards are discarded for usage with computers. Taking the advantage of simplicity of PS2 protocol, a wireless keyboard is designed using Arduino, HC-05 bluetooth and PS/2 keyboard. The keystrokes thus received by the computer using bluetooth dongle are made to print on the Notepad of the computer.
